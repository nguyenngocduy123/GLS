How to authenticate?

1. Request for API token

API Documentation: Log in

Repeat the same steps to get a new token

Request

    // POST http://localhost/api/auth/v2/login
    {
        "username": "user_doe",
        "password": "123"
    }

Response

Subsequent requests MUST provide token in the request header[x-access-token]

Each token is valid for (5) days

Do not take it for granted that the API will always return status 200

    // HTTP/1.1 200 OK
    {
        "_id": "58118ee072e0c18ab626501e",
        "username": "user_doe",
        "fullname": "Planner Doe",
        "role": "planner"
        "usergroup": null,
        "forceChangePassword": true,
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9"
    }
