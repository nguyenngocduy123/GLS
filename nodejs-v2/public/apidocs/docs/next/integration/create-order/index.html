How to create order?

1. Log into the system

API Documentation: Log in

Request

User role must be a planner

    // POST http://localhost/api/auth/v2/login
    {
        "username": "user_doe",
        "password": "123"
    }

Response

Subsequent requests MUST provide token in the request header[x-access-token]

    // HTTP/1.1 200 OK
    {
        "_id": "58118ee072e0c18ab626501e",
        "username": "user_doe",
        "fullname": "Planner Doe",
        "role": "planner"
        "usergroup": null,
        "forceChangePassword": true,
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9"
    }

2. Create an order

API Documentation: Create Order

Request

For example,

  • ABC Company Pte. Ltd. requests for a pickup at 11 Tuas Link, Singapore 639588.
  • The (pickup) job is low priority (i.e. 1).
  • The job needs to be done on 2018-01-01, between 9 AM to 1 PM, +08:00 Singapore time.
  • It will take approximately 15 minutes to fulfill the job.
  • The job will require one special equipment - Crane, to pickup 5 Bag.
  • The person-in-charge (i.e. contact person) at 11 Tuas Link 1 is John Doe.
  • Upon reaching, John Doe must provide a verification code 12345 to ensure that the driver is at the correct place.
    // header['x-access-token'] = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9'
    // POST http://localhost/web/data/v2.1/order
    {
        "record": [{
            "Id": "sg-12345",
            "Priority": 1,
            "CustomerName": "ABC Company Pte. Ltd.",
            "CustomerPhone": 66123456,
            "CustomerEmail": "example@abc.com",
            "VehicleRestriction": null,
            "DeliveryDetails": [{
                "JobSequence": 1,
                "JobType": "PICKUP",
                "Address": "11 Tuas Link 1",
                "Postal": "639588",
                "Lat": 1.3341,
                "Lng": 103.636,
                "StartTimeWindow": "2018-01-01T09:00:00+08:00",
                "EndTimeWindow": "2018-01-01T13:00:00+08:00",
                "ServiceTime": 15,
                "NoteFromPlanner": [
                    { key: "Equipments", value: "Crane" }
                ],
                "ContactName": "John Doe",
                "ContactPhone": 91234567,
                "ContactEmail": null,
                "DeliveryItems": [
                    { "ItemId": "Bag", "ItemQty": 5 }
                ],
                "VerificationCode": {
                    "Code": "12345"
                }
            }]
        }]
    }

Response

    // HTTP/1.1 200 OK
    ["sg-12345"] // one record inserted
