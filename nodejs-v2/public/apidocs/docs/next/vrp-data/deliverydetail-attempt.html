PUT /api/data/v2.1/job/:jobId/attempt

Indicates that a job was attempted by driver, regardless successful or unsuccessful

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleDriver
Rate limited?No
CSRF enabledYes

Request Params

FieldRequiredTypeDefault ValueDescription
jobIdrequiredIntegerId column of DeliveryDetail table

Request Body

Encoding: multipart/form-data

FieldRequiredTypeDefault ValueDescription
deliveredrequiredBooleanIndicates whether the delivery was successful or unsuccessful.
noterequiredNote[]Remarks/memo from driver
signaturerequiredFileProof of delivery - Signature in image/png, image/jpg or image/jpeg. Set only if delivered=true
photooptionalFileProof of delivery - Photo in image/png, image/jpg or image/jpeg. Set only if delivered=true
notePhotosoptionalFile[]List of note photos in image/png, image/jpg or image/jpeg. Maximum (3) photos
itemsoptionalObject[]List of items picked-up or delivered. Set only if delivered=true
items[].IdoptionalIntegerId of job item that was picked-up or delivered
items[].ActualItemQtyoptionalFloatQuantity that was picked-up or delivered.

Request Example

Successful job attempt

    // http://localhost/api/data/v2.1/job/100/attempt
    {
        "delivered": true,
        "note": [
            { "key": "template", "value": null },
            { "key": "comments", "value": "Waited 10 minutes" }
        ],
        "signature": (BLOB),
        "photo": (BLOB),
        "notePhotos": [(BLOB), (BLOB)],
        "items": [
            { "Id": 1, "ActualItemQty": 15 }
        ]
    }

Unsuccessful job attempt

    // http://localhost/api/data/v2.1/job/100/attempt
    {
        "delivered": false,
        "note": [
            { "key": "template", "value": "Customer Not Around" }
        ],
        "notePhotos": [(BLOB), (BLOB)]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredDeliveryDetailUpdated job details

Success Response Example

Note: Some details are removed for readability

    // http://localhost/api/data/v2.1/job/100/attempt
    {
        "Id": 100,
        "DeliveryMasterId": "sg-12345",
        "Status": 3,
        "NoteFromDriver": [
            { "key": "template", "value": null },
            { "key": "comments", "value": "Waited 10 minutes" }
        ],
        "ActualDeliveryTime": "2018-01-01T09:30:00Z",
        "DeliveryMaster": {
            "VehicleId": "Truck3Ft",

            "LastAttemptedByPlateNumber": "SJJ1234M"
        },
        "DeliveryItems": [{
            "Id": 1,
            "ActualItemQty": 15
        }]
        // ...
    }

PUT /api/data/v2.1/job/:jobId/attemptB64

Copy of original attempt API. Reads images in Base64 instead of Blob

This API will deprecate anytime. Do not use

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleDriver
Rate limited?No
CSRF enabledYes

Request Params

FieldRequiredTypeDefault ValueDescription
jobIdrequiredIntegerId column of DeliveryDetail table

Request Body

Similar to original attempt API except images fields

FieldRequiredTypeDefault ValueDescription
signaturerequiredStringProof of delivery - Signature in image/png, image/jpg or image/jpeg encoded in Base64. Set only if delivered=true
photooptionalStringProof of delivery - Photo in image/png, image/jpg or image/jpeg encoded in Base64. Set only if delivered=true
notePhotosoptionalString[]List of note photos in image/png, image/jpg or image/jpeg encoded in Base64. Maximum (3) photos

Request Example

Successful job attempt

    // http://localhost/api/data/v2.1/job/100/attempt
    {
        "delivered": true,
        "note": [
            { "key": "template", "value": null },
            { "key": "comments", "value": "Waited 10 minutes" }
        ],
        "signature": "R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7",
        "photo": "R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7",
        "notePhotos": ["R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7"],
        "items": [
            { "Id": 1, "ActualItemQty": 15 }
        ]
    }

Success Response

Refer to original attempt API for success response

POST /api/data/v2.1/job/sync

Synchronises job attempt files that are created when driver has no Internet connection.

Invalid files are ignored and deleted from server

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner, Driver
Rate limited?No
CSRF enabledYes

Request Body

Encoding: multipart/form-data

FieldRequiredTypeDefault ValueDescription
[]requiredObject[]List of jobs to synchronise
[].jobIdrequiredStringId of job that was attempted
[].deliveredrequiredBooleanIndicates whether the delivery was successful or unsuccessful.
[].attemptedByDriverrequiredStringFull name of driver who attempted this job
[].attemptedByVehiclerequiredStringPlate number of the vehicle assigned to the driver
[].deliveryTimerequiredStringActual time of delivery in ISO8601 format
[].noterequiredNote[]Remarks/memo from driver
[].signaturerequiredFileProof of delivery - Signature in image/png, image/jpg or image/jpeg. Set only if delivered=true
[].photooptionalFileProof of delivery - Photo in image/png, image/jpg or image/jpeg. Set only if delivered=true
[].notePhotosoptionalFile[]List of note photos in image/png, image/jpg or image/jpeg. Maximum (3) photos
[].itemsoptionalObject[]List of items picked-up or delivered. Set only if delivered=true
[].items[].IdoptionalIntegerId of job item that was picked-up or delivered
[].items[].ActualItemQtyoptionalFloatQuantity that was picked-up or delivered.

Request Example

    // http://localhost/api/data/v2.1/job/sync
    [{
        // successful job attempt
        "jobId": 100,
        "delivered": true,
        "attemptedByDriver": "Driver Doe",
        "attemptedByVehicle": "SJJ1234M",
        "deliveryTime": "2018-01-01T09:30:00+08:00",
        "note": [
            { "key": "template", "value": null },
            { "key": "comments", "value": "Waited 10 minutes" }
        ],
        "signature": (BLOB),
        "photo": (BLOB),
        "notePhotos": [(BLOB), (BLOB)],
        "items": [
            { "Id": 1, "ActualItemQty": 15 }
        ]
    }, {
        // unsuccessful job attempt
        "jobId": 101,
        "delivered": false,
        "attemptedByDriver": "Driver Doe",
        "attemptedByVehicle": "SJJ1234M",
        "deliveryTime": "2018-01-01T09:50:00+08:00",
        "note": [
            { "key": "template", "value": "Customer Not Around" }
        ],
        "notePhotos": [(BLOB), (BLOB)]
    }]

Success Response

HTTP Status 204
