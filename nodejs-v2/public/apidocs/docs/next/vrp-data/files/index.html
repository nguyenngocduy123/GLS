GET /web/data/v2.1/files/transactionLog

Generate excel file for transactions that happen during a period of time

Resource Info

Information
Response formatapplication/zip
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Query String

One of the date filters must be present when querying. (i.e. not allowed to query all records from database)

FieldRequiredTypeDefault ValueDescription
startDateoptionalStringGet jobs within a date range (startDate, endDate). Value must be in ISO8601 format
endDateoptionalStringstartDateGet jobs within a date range (startDate, endDate). Value must be in ISO8601 format
dateoptionalStringGet jobs on a date. Value must be in ISO8601 format
withPhotosoptionalBooleanfalseDownload all photos associated with the jobs as well. Maximum number of jobs can be queried is (5000)

Request Example

  • Download transaction log for today
    • http://localhost/web/data/v2.1/files/transactionLog?date=2018-01-01
  • Download transaction log for orders between 2018-01-01 to 2018-01-31
    • http://localhost/web/data/v2.1/files/transactionLog?startDate=2018-01-01&endDate=2018-01-31
  • Download transaction log for today with all photos
    • http://localhost/web/data/v2.1/files/transactionLog?date=2018-01-01&withPhotos=true

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredBufferZip file
requiredExcelTransaction log in .xlsx
Note PhotosoptionalFolderAll note photos associated to jobs in the transaction log
Proof of DeliveryoptionalFolderAll proof of delivery associated to jobs in the transaction log
