POST /web/data/v2.1/vehicleType

Create vehicle type(s). If any vehicle type fails to create, the entire list of vehicle types will not be created

If you do not know what values to input, the following values are suitable for most cases:

  • FixedCost = 100
  • DistanceCost = 0.0001
  • TravelTimeCost = 0.0017
  • WaitingTimeCost = 0.0017

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
recordrequiredObject[]List of vehicle types to insert into the database
record[].NamerequiredString(...50)Vehicle type identifier
record[].CapacityoptionalFloat0Total weight that can be carried by the vehicle
record[].FixedCostoptionalFloat0Daily one-time fixed cost of activating a vehicle
record[].DistanceCostoptionalFloat0Cost incurred per meter travelled by vehicle
record[].TravelTimeCostoptionalFloat0Cost incurred per second spent by vehicle travelling
record[].WaitingTimeCostoptionalFloat0Cost incurred per second spent by driver waiting to serve the customer

Request Example

    // http://localhost/web/data/v2.1/vehicleType
    {
        "record": [{
            "Name": "Van",
            "Capacity": 1000,
            "FixedCost": 100,
            "DistanceCost": 0.0001,
            "TravelTimeCost": 0.0017,
            "WaitingTimeCost": 0.0017
        }]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredString[]List of Id of inserted records

Success Response Example

    // HTTP/1.1 200 OK
    [200] // one record inserted

newValues.DistanceCostoptionalFloat0Cost incurred per meter travelled by vehicle
