Verification Code

Table: VerificationCode

Reference

Prefix the URLs with the server URL.

For example:

/web/data/v2.1/example/:id => http://localhost/web/data/v2.1/example/:id

I want to...CSRF EnabledAPI
Create/Update verification code for a jobYesPUT /web/data/v2.1/code/:jobId
Delete verification code of a jobYesDELETE /web/data/v2.1/code/:jobId
Get verification code of a jobYesGET /web/data/v2.1/code/:jobId
Notify verification codeYesPOST /web/data/v2.1/code/:jobId/notify

PUT /web/data/v2.1/code/:jobId

Create/update (upsert) verification code for a job - i.e. if verification code already exists for a job, overwrite the existing value.

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Params

FieldRequiredTypeDefault ValueDescription
jobIdrequiredIntegerId column of DeliveryDetail table

Request Body

FieldRequiredTypeDefault ValueDescription
newValuesrequiredObjectValues to overwrite existing record with
newValues.CoderequiredString(5)Actual verification code value. Numbers only

Request Example

    // create/update verification code of job 100
    // http://localhost/web/data/v2.1/code/100
    {
        "newValues": {
            "Code": "12345"
        }
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredIntegerNumber of rows upserted

Success Response Example

    // HTTP/1.1 200 OK
    1 // one record inserted
