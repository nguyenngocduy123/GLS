POST /api/message/v2

Create message(s). Messages will be created as many as possible (i.e. if any fails, other messages will still be persisted)

New message of a job will always overwrite the old message of the same job

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleDriver
Rate limited?No
CSRF enabledNo

Request Body

FieldRequiredTypeDefault ValueDescription
jobsrequiredDeliveryDetail[]List of jobs that share the same message
jobs[].UserGrouprequiredStringUsergroup of the driver
lateByrequiredStringDuration that driver will be late for
driverRemarksoptionalStringRemarks from driver

Request Example

    // http://localhost/api/message/v2
    {
        "jobs": [{
            "Id": 100,
            "DeliveryMasterId": "sg-12345",
            "Status": 1,
            "JobSequence": 1,
            "JobType": "PICKUP",
            "Address": "11 Tuas Link 1",
            "Postal": "639588",
            "Lat": 1.3341,
            "Lng": 103.636,
            "StartTimeWindow": "2018-01-01T09:00:00Z",
            "EndTimeWindow": "2018-01-01T13:00:00Z",
            "ServiceTime": 15,
            "NoteFromPlanner": null,
            "NoteFromDriver": null,
            "ContactName": null,
            "ContactPhone": null,
            "ContactEmail": null,
            "EngineRouteSeqNum": 11,
            "ActualDeliveryTime": null,
            "UserGroup": null
        }],
        "lateBy": '30 Minutes',
        "driverRemarks": 'Traffic jam'
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredInteger[]List of job ids of inserted messages

Success Response Example

    // http://localhost/api/message/v2
    [100]

PUT /web/message/v2/process/:messageId

Acknowledge / process a message

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Params

FieldRequiredTypeDefault ValueDescription
messageIdrequiredStringMessage Id

Request Example

  • Acknowledge message
    • http://localhost/web/message/v2/process/5bac93582c431fbef17d4b08

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
_idrequiredStringMessage Id
jobrequiredDeliveryDetailJob details
job.UserGrouprequiredStringUsergroup of the driver
lateByrequiredStringDuration that driver will be late for
driverRemarksoptionalStringRemarks from driver
fromUsernamerequiredStringUsername of driver
fromUseroptionalObjectDriver details
fromUser.fullnameoptionalStringFull name of driver
fromUser.phoneoptionalIntegerContact number of driver
processedByUsernamerequiredStringUsername of the user who has acknowledged the message
processedByoptionalObjectDetails of the user who has acknowledged the message
processedBy.fullnameoptionalStringFull name of the user who has acknowledged the message
processedBy.phoneoptionalIntegerContact number of the user who has acknowledged the message
created_daterequiredStringDate and time when the message was created. Value is in ISO8601 format
modified_daterequiredStringDate and time of the last message update. Value is in ISO8601 format
toUserRolesrequiredString[]List of roles that the message is targeted to

GET /web/message/v2

Get list of messages based on date filter (optional)

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Query String

FieldRequiredTypeDefault ValueDescription
startDateoptionalStringGet jobs after a certain date. Value must be in ISO8601 format
endDateoptionalStringGet jobs before a certain date. Value must be in ISO8601 format

Request Example

  • Get all messages
    • http://localhost/web/message/v2
  • Get messages after a date
    • http://localhost/web/message/v2?startDate=2018-01-01
  • Get messages before a date
    • http://localhost/web/message/v2?endDate=2018-01-31
  • Get messages between 1 January 2018 and 31 Feburary 2018
    • http://localhost/web/message/v2?startDate=2018-01-01&endDate=2018-01-31

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
[]requiredMessage[]List of messages

Success Response Example

    // http://localhost/web/message/v2
    [{
        "_id": "5bac93582c431fbef17d4b08",
        "job": {
            "Id": 100,
            "DeliveryMasterId": "sg-12345",
            "UserGroup": null
            // ...
        },
        "lateBy": '30 Minutes',
        "driverRemarks": 'Traffic jam',
        "fromUsername": "driver_doe",
        "fromUser": {
            "fullname": "Driver Doe",

        },
