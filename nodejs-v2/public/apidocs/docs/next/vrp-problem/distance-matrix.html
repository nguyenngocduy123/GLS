Prefix the URLs with the server URL.

For example:

/web/data/v2.1/example/:id => http://localhost/web/data/v2.1/example/:id

Generate distance matrix for a problemYesGET /web/problem/v2/distanceMatrix/:problemId

GET /web/problem/v2/distanceMatrix/:problemId

Generate distance matrix in .csv format for a problem based on problem.addresses

Use Geocoding API to populate problem.addresses
