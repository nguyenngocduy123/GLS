Geocode

Reference

Prefix the URLs with the server URL.

For example:

/web/data/v2.1/example/:id => http://localhost/web/data/v2.1/example/:id

I want to...CSRF EnabledAPI
Geocode a problemYesPOST /web/problem/v2/geocode/:problemId

POST /web/problem/v2/geocode/:problemId

Populate problem.addresses based on the addresses used in the problem

Refer to Map Service > Geocode for additional information

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin, Default, Planner
Rate limited?No
CSRF enabledYes

Request Query String

FieldRequiredTypeDefault ValueDescription
servicerequiredStringonemapMap service preferred for database values. Accepted values: ONEMAP, SDMAP
alternativeServicesoptionalStringComma-delimited string. List of alternative cloud map services. Accepted values: ONEMAP, SDMAP
saveToDboptionalBooleanfalsetrue if save the coordinates that were queried from cloud services into database

Request Example

  • Populate addresses using onemap
    • http://localhost/web/problem/v2/geocode/5bb1c088f52ea34c8949bff5?service=onemap
  • Populate addresses and save queried coordinates to database
    • http://localhost/web/problem/v2/geocode/5bb1c088f52ea34c8949bff5?saveToDb=true

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
_idrequiredStringProblem identifier

addressesrequiredAddress[]List of addresses in the problem
modified_daterequiredStringDate of last modification in ISO8601 format
created_daterequiredStringDate of creation in ISO8601 format
