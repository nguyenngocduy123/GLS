POST /web/problem/v2

Create a problem

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin, Default, Planner
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
namerequiredStringp_MMDDYYYY[_]HHmmName of problem
fleet_sizerequiredStringFINITEWhether number of vehicles available for optimisation. Accepted values: FINITE, INFINITE
FINITE - number of vehicles are predefined and fixed
INFINITE - number of vehicles cannot be determined before optimising
coord_moderequiredStringREALTypes of coordinates used. Accepted values: REAL, 2D
REAL - Use latitudes and longitudes and real distances (for real-world problem)
2D - Use x/y and Cartesian distances (for research)
usernameoptionalString[]List of usernames that are allowed to read this problem
vehicle_typesrequiredVehicleType[]List of vehicle types that each vehicle is assigned to
vehiclesrequiredVehicles[]List of vehicles to assign jobs to
servicesrequiredService[]List of single jobs (either pickup or delivery)
shipmentsrequiredShipment[]List of double jobs (pickup with delivery)
solutionsrequiredSolution[]Possible solutions for this problem

PATCH /web/problem/v2/:problemId/update

Update any property in a problem based on filter. The property can be root or a nested property

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin, Default, Planner
Rate limited?No
CSRF enabledYes

Request Params

FieldRequiredTypeDefault ValueDescription
problemIdrequiredInteger_id of problem

Request Body

FieldRequiredTypeDefault ValueDescription
updaterequiredObjectNew values
queryoptionalObjectFilter to select the document to update

Request Example

Update root property

    // http://localhost/web/problem/v2/5bb1c088f52ea34c8949bff5/update
    {
        "update": {
            "coord_mode": "INFINITE"
        }
    }

Update object in an array property

    // http://localhost/web/problem/v2/5bb1c088f52ea34c8949bff5/update
    {
        "update": {
            "solutions.$": { // entire object will be overwritten
                "id": "solution 2",
                "routes": [{
                    "vehicle_id": "vehicle 1",
                    "act": [{
                        "type": "service",
                        "job_id": "order 1",
                        "job_prop": "delivery"
                    }, {
                        "type": "shipment",
                        "job_id": "order 2",
                        "job_prop": "pickup"
                    }, {
                        "type": "shipment",
                        "job_id": "order 2",
                        "job_prop": "delivery"
                    }]
                }]
            }
        },
        "query": {
            "solutions.id": "solution 1"
        }
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
_idrequiredStringProblem identifier
requiredProblemUpdated problem

Success Response Example

    // http://localhost/web/problem/v2/5bb1c088f52ea34c8949bff5/update
    {
        "_id": "5bb1c088f52ea34c8949bff5",
        "fleet_size": "INFINITE",
        // ...
    }

PATCH /web/problem/v2/:problemId/remove

Remove element(s) in an array in a problem based on filter

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin, Default, Planner
Rate limited?No
CSRF enabledYes

Request Params

FieldRequiredTypeDefault ValueDescription
problemIdrequiredInteger_id of problem

Request Body

FieldRequiredTypeDefault ValueDescription
updaterequiredObjectFilter to decide how to remove element in array property

Request Example

    // http://localhost/web/problem/v2/5bb1c088f52ea34c8949bff5/remove
    {
        "update": {
            "solutions": {
                "id": {
                    "$in": ["solution 1"]
                }
            }
        }
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
_idrequiredStringProblem identifier
requiredProblemUpdated problem

Success Response Example

    // http://localhost/web/problem/v2/5bb1c088f52ea34c8949bff5/remove
    {
        "_id": "5bb1c088f52ea34c8949bff5",
        "solutions": []
        // ...
    }

GET /web/problem/v2/abstract

Get brief summary of each solution in the database

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin, Default, Planner
Rate limited?No
CSRF enabledYes

Request Example

  • Get abstract information / brief summary
    • http://localhost/web/problem/v2/abstract

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
[]requiredStringList of problems in the database
[]._idrequiredStringProblem identifier
[].namerequiredStringName of problem
[].fleet_sizerequiredStringWhether number of vehicles available for optimisation
[].coord_moderequiredStringTypes of coordinates used
[].no_of_ordersrequiredIntegerNumbers of orders (service + shipment) in the problem
[].no_of_vehiclesrequiredIntegerNumbers of vehicles in the problem
[].no_of_solutionsrequiredIntegerNumbers of solutions in the problem
[].modified_daterequiredStringDate of last modification in ISO8601 format
[].created_daterequiredStringDate of creation in ISO8601 format
