Route

costs, time and distance values are for this particular trip/route only.

FieldRequiredTypeDefault ValueDescription
vehicle_idrequiredStringVehicle assigned to this route
fixed_costsrequiredFloatUnit: Dollars. Total fixed cost
variable_costsrequiredFloatUnit: Dollars. Total variable (dependent) cost
distancerequiredFloatUnit: Meters. Total distance covered
transport_timerequiredFloatUnit: Dollars. Total travel time
no_activitiesrequiredIntegerTotal number of jobs to do in this route
start_timerequiredStringEstimated start time of route in YYYY-MM-DDTHH:mmZ format
end_timerequiredStringEstimated end time of route in YYYY-MM-DDTHH:mmZ format
max_loadrequiredInteger[]If engine value is 'chinh', max_load parameter supports an array of values. If engine value is 'siwei', max_load parameter is a single integer
actrequiredAct[]List of jobs to do for this route
