POST /web/auth/v2/login

Logs non-driver users into the system. For drivers, see POST /api/auth/v2/login.

Only one active session allowed per user.

  • For administrators, the previous user will be forced to log out.
  • For other users, API will return error response 500.

Account lockout is enabled after (5) consecutive unsuccessful logins.

Account lockout period is (30) seconds.

Resource Info

Authorised RolePlanner, Default, Admin

CSRF enabledYes

POST /api/auth/v2/login

Logs planner or driver users into the system. For other users, see POST /web/auth/v2/login.

Only one active session allowed per user.

  • For administrators, the previous user will be forced to log out.
  • For other users, API will return error response 500.

Account lockout is enabled after (5) consecutive unsuccessful logins.

Account lockout period is (30) seconds.

Authorised RolePlanner, Driver

CSRF enabledNo
