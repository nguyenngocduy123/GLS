Prefix the URLs with the server URL.

For example:

/web/data/v2.1/example/:id => http://localhost/web/data/v2.1/example/:id

Enable usersYesPATCH /web/user/v2/enable

Force logout usersYesPOST /web/user/v2/forceLogout

PATCH /web/user/v2/enable

Enable user(s)

Resource Info

Information
Response formatText
Requires authentication?Yes
Authorised RoleAdmin
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
requiredString[]List of usernames of accounts to enable

Request Example

    // http://localhost/web/user/v2/enable
    ["user_doe"]

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredIntegerNumber of users enabled

Success Response Example

    // http://localhost/web/user/v2/enable
    1 // one user enabled

POST /web/user/v2/forceLogout

Force log out a user

Users are not allowed to have multiple simultaneous login. This API purges the existing session

Resource Info

Information
Response formatText
Requires authentication?Yes
Authorised RoleAdmin
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
usernamesrequiredString[]List of usernames of accounts to force log out

Request Example

    // http://localhost/web/user/v2/forceLogout
    {
        "usernames": ["user_doe"]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredIntegerNumber of users forced logged out

Success Response Example

    // http://localhost/web/user/v2/forceLogout
    1 // one user was active and forced to log out

POST /web/user/v2/resetPassword

Reset password for any user in the system

Depending on the system, some may reset the password and notify the password to the user via email

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
usernamesrequiredString[]List of usernames of accounts to reset password for

Request Example

    // http://localhost/web/user/v2/resetPassword
    {
        "usernames": ["user_doe"]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredIntegerNumber of users whose password are reset successfully

Success Response Example

    // http://localhost/web/user/v2/resetPassword
    1 // one user password has been reset
