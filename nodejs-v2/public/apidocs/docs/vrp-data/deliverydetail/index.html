POST /web/data/v2.1/job

Create job(s) for any orders. If any job fails to create, the entire list of jobs will not be created

The following columns cannot be overwritten with this API:

  • Status
    • Default: 1 (i.e. PENDING)
  • ActualDeliveryTime
    • Default: null
  • EngineRouteSeqNum
    • Default: null
  • NoteFromDriver
    • Default: null

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
recordrequiredObject[]List of jobs to insert into the database
record[].DeliveryMasterIdrequiredString(...30)Order Id that the job belongs to
record[].JobSequencerequiredInteger0Indicates the execution order for jobs for each order
record[].JobTyperequiredStringIndicates pick-up or delivery job. Allowed values: PICKUP, DELIVERY
record[].AddressrequiredString(...1024)Street address
record[].PostalrequiredString(6)Postal code of the address
record[].LatrequiredFloatLatitude of Postal in degrees
record[].LngrequiredFloatLongitude of Postal in degrees
record[].StartTimeWindowrequiredStringStart time of time window in ISO8601 format
record[].EndTimeWindowrequiredStringEnd time of time window in ISO8601 format
record[].ServiceTimerequiredFloat15Estimated time required to finish the job (e.g. time taken to unload goods)
record[].NoteFromPlanneroptionalNote[]Remarks/memo from planner
record[].ContactNameoptionalString(...254)Name of person to contact at Address
record[].ContactPhoneoptionalString(8)Phone number of person to contact at Address. Numbers only
record[].ContactEmailoptionalString(...254)Email address of person to contact at Address
record[].DeliveryItemsoptionalDeliveryItem[]List of items to pick-up or delivery for the job
record[].VerificationCodeoptionalVerificationCodeVerification code for the job

Request Example

    // http://localhost/web/data/v2.1/job
    {
        "record": [{
            "DeliveryMasterId": "sg-45678",
            "JobSequence": 1,
            "JobType": "DELIVERY",
            "Address": "11 Tuas Link 1",
            "Postal": "639588",
            "Lat": 1.3341,
            "Lng": 103.636,
            "StartTimeWindow": "2018-01-01T14:00:00+08:00",
            "EndTimeWindow": "2018-01-01T15:00:00+08:00",
            "ServiceTime": 15,
            "NoteFromPlanner": [
                { key: "Payment", value: "Cash" }
            ],
            "ContactName": "Jane Doe",
            "ContactPhone": 97654321,
            "ContactEmail": null,
            "DeliveryItems": [
                { "ItemId": "Bag", "ItemQty": 5 }
            ],
            "VerificationCode": {
                "Code": "12345"
            }
        }]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredString[]List of Id of inserted records

Success Response Example

    // HTTP/1.1 200 OK
    ["2"] // one record inserted

GET /web/data/v2.1/job

Get list of jobs based on either, where filter or date filter

  • Planners will get all jobs according to the filters
  • Drivers will only get jobs that are assigned to his vehicles after filtering

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner, Driver
Rate limited?No
CSRF enabledYes

Request Query String

One of the following filters must be present when querying. (i.e. not allowed to query all records from database)

FieldRequiredTypeDefault ValueDescription
whereoptionalStringWhere conditions in JSON string
where.DeliveryMasteroptionalStringWhere conditions for orders/DeliveryMaster in JSON string
startDateoptionalStringGet jobs within a date range (startDate, endDate). Value must be in ISO8601 format
endDateoptionalStringGet jobs within a date range (startDate, endDate). Value must be in ISO8601 format
dateoptionalStringGet jobs on a date. Value must be in ISO8601 format

Request Example

  • Get jobs at a location
    • http://localhost/web/data/v2.1/job?where={"Postal":"639588"}
  • Get jobs assigned to vehicle Truck3Ft
    • http://localhost/web/data/v2.1/job?where={"DeliveryMaster":{"VehicleId":""VehicleId": "Truck3Ft""}}
  • Get jobs on 1 January 2018
    • http://localhost/web/data/v2.1/job?date=2018-01-01
  • Get jobs between 1 January 2018 and 31 Feburary 2018
    • http://localhost/web/data/v2.1/job?startDate=2018-01-01&endDate=2018-02-31

HTTP Status 200
