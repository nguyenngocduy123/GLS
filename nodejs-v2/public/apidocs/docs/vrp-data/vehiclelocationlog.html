GET /web/data/v2.1/vehicleLog/last

Get the last seen location of all vehicles

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Example

  • Get last seen vehicle location logs
    • http://localhost/web/data/v2.1/vehicleLog/last

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
[]requiredObject[]List of vehicle location logs
[].VehicleIdrequiredStringId of vehicle
[].VehicleUserGrouprequiredStringUsergroup that vehicle was assigned with
[].DriverNamerequiredStringUsername of the driver that was assigned to the vehicle
[].PlateNumberrequiredStringPlate number of the vehicle that driver was assigned with
[].LatrequiredFloatLatitude in degrees
[].LngrequiredFloatLongitude in degrees
[].AccuracyrequiredFloatAccuracy of coordinates
[].RecordedTimerequiredStringTime that coordinates was recorded on. Value must be in ISO8601 format

Success Response Example

    // http://localhost/rest/data/v2.1/vehicleLog/last
    [{
        "VehicleId": "Truck3Ft",
        "VehicleUserGroup": null,
        "DriverName": "Driver Doe",
        "PlateNumber": "SJJ1234M",
        "Lat": 1.3553794,
        "Lng": 103.5677444,
        "Accuracy": 50,
        "RecordedTime": "2018-01-01T09:00:00+08:00"
    }, {
        // ...
    }]

POST /rest/data/v2.1/vehicleLog

Create vehicle log(s). Vehicle logs will be created as many as possible (i.e. invalid logs will not cause error to return)

Resource Info

Information
Response formatJSON
Requires authentication?Yes, strictly by token only
Authorised RoleDriver
Rate limited?No
CSRF enabledNo

Request Body

FieldRequiredTypeDefault ValueDescription
recordrequiredObject[]List of orders to insert into the database
record[].VehicleIdrequiredString(...50)Id of vehicle
record[].VehicleUserGrouprequiredString(...100)Usergroup that vehicle was assigned with
record[].DriverNamerequiredString(...100)Username of the driver that was assigned to the vehicle
record[].PlateNumberrequiredString(...10)Plate number of the vehicle that driver was assigned with
record[].LatrequiredFloatLatitude in degrees
record[].LngrequiredFloatLongitude in degrees
record[].AccuracyrequiredFloatAccuracy of coordinates
record[].RecordedTimerequiredStringTime that coordinates was recorded on. Value must be in ISO8601 format

Request Example

    // http://localhost/rest/data/v2.1/vehicleLog
    {
        "record": [{
            "VehicleId": "Truck3Ft",
            "VehicleUserGroup": null,
            "DriverName": "Driver Doe",
            "PlateNumber": "SJJ1234M",
            "Lat": 1.3553794,
            "Lng": 103.5677444,
            "Accuracy": 50,
            "RecordedTime": "2018-01-01T09:00:00+08:00"
        }]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredIntegerNumber of inserted records

Success Response Example

    // HTTP/1.1 200 OK
    1 // one record inserted
