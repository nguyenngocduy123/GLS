GET /web/map/v2/searchAddress

Search for similar addresses based on either postal code or address

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner, Default, Admin
Rate limited?No
CSRF enabledYes

Request Query String

Either postal or address must be specified

FieldRequiredTypeDefault ValueDescription
postaloptionalStringSubstring of any postal code
addressoptionalStringSubstring of any address
servicerequiredStringonemapMap service. Accepted values: (postal) ONEMAP, SDMAP. (address) ONEMAP, GMAP, BDMAP.

Request Example

  • Query for postal codes that contains 123
    • http://localhost/web/map/v2/searchAddress?postal=123
  • Query for address that contains Fusionopolis Way
    • http://localhost/web/map/v2/searchAddress?address=Fusionopolis%20Way
  • Query for postal using onemap
    • http://localhost/web/map/v2/searchAddress?postal=123&service=onemap

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredObject[]List of similar addresses
[].postaloptionalStringPostal code provided by map service. Returned when querying for postal
[].full_addressoptionalStringAddress specified in query string. Returned when querying for address
[].addressoptionalStringFull address provided by map service. Returned when querying for address
[].service-latrequiredFloatLatitude provided by map service
[].service-lonrequiredFloatLongitude provided by map service

Success Response Example

    // http://localhost/web/map/v2/searchAddress?address=Fusionopolis%20Way
    [{
        "full_address": "Fusionopolis Way",
        "address": "FUSIONOPOLIS WAY",
        "onemap-lat": 1.29936381703016,
        "onemap-lon": 103.788594134718
    }]

POST /web/map/v2/geocode

Add coordinates to a list of addresses

It will query database for postal code, if postal code does not exist, then it will query cloud services

User inserted postal codes will always take precedence over all other postal code sources

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner, Default, Admin
Rate limited?No
CSRF enabledYes

Request Query String

FieldRequiredTypeDefault ValueDescription
servicerequiredStringonemapMap service preferred for database values. Accepted values: ONEMAP, SDMAP
alternativeServicesoptionalStringComma-delimited string. List of alternative cloud map services. Accepted values: ONEMAP, SDMAP
saveToDboptionalBooleanfalsetrue if save the coordinates that were queried from cloud services into database

Request Body

FieldRequiredTypeDefault ValueDescription
addressesrequiredObject[]List of postal codes
addresses[].postalrequiredStringPostal code

Request Example

    // http://localhost/web/map/v2/geocode
    {
        "addresses": [{
            "postal": "730653",
        }]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredObject[]List of similar addresses
[].postalrequiredStringPostal code
[].latrequiredFloatLatitude of postal in degrees
[].lonrequiredFloatLatitude of postal in degrees

Success Response Example

    // http://localhost/web/map/v2/geocode
    [{
        "postal": "730653",
        "lat": 1.43808142338,
        "lon": 103.799032334
    }]

POST /map/v2/route

Get route information such as travel distance and time between pairs of coordinates

Resource Info

Information
Response formatJSON
Requires authentication?No
Authorised Role
Rate limited?No
CSRF enabledNo

Request Query String

FieldRequiredTypeDefault ValueDescription
servicerequiredStringosmapMap service. Accepted values: OSMAP, GHMAP.

Request Body

FieldRequiredTypeDefault ValueDescription
coordinatesrequiredObject[]List of coordinate pairs. At least (2) pairs are required
coordinates[].[]requiredFloat[]One array represents one pair of coordinates. First element is latitude, second element is longitude

Request Example

    // http://localhost/map/v2/route
    {{
        "coordinates": [
            [1.353016,103.757745],
            [1.359709,103.767605]
        ]
    }

Success Response

HTTP Status 200

If service is osmap, refer to OSRM API for further description

FieldRequiredTypeDefault ValueDescription
coderequiredStringOkResponse code from map service
routesrequiredObject[]List of route through multiple waypoints
routes[].legsrequiredRouteLeg[]Route legs between the given waypoints
routes[].geometryrequiredStringGeometry of the route value (not simplified)
routes[].durationrequiredFloatEstimated travel time, in float number of seconds
routes[].distancerequiredFloatDistance traveled by the route, in float meters
waypointsrequiredWaypoint[]Waypoint on a route
