POST /web/map/v2/coords

Create/update (upsert) a user-specified postal code

Resource Info

Information
Response formatText
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Body

The coordinates must be within the area of Singapore, Jakarta, Shanghai and Kuching.

FieldRequiredTypeDefault ValueDescription
addressesrequiredObject[]Details of postal code
addresses[].postalrequiredString(6)Postal code
addresses[].latrequiredFloatLatitude of postal in degrees
addresses[].lngrequiredFloatLongitude of postal in degrees

Request Example

    // http://localhost/web/map/v2/coords
    {
        "addresses": [{
            "postal": "123123",
            "lat": "1.446151",
            "lng": "103.815652"
        }]
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
requiredIntegerNumber of postal codes upserted

Success Response Example

    // http://localhost/web/map/v2/coords
    1 // one record inserted
