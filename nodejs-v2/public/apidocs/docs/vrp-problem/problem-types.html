Vehicle Restriction

PrefixPurpose
(None)Allowed vehicles
#Not allowed vehicles
@Allowed vehicle types
#@Not allowed vehicle types

Vehicle is identified by its Id field

Vehicle type is identified by its Name field

Examples:

  • Truck3Ft (Plate Number SJJ1234M)
    • Vehicle Truck3Ft is allowed to do the order
  • @10Feet
    • Vehicles that are under the category of 10Feet are allowed to do the order
  • #SJJ4567M
    • Vehicle SJJ4567M is not allowed to do the order
  • #@Van
    • Vehicles that are under the category of Van are not allowed to do the order

Problem

Vehicle Type

FieldRequiredTypeDefault ValueDescription
idrequiredStringUnique id of vehicle type
capacityrequiredFloat[]Each element in the array represents an job item. The element value indicates the maximum capacity that each vehicle can carry for that particular item
fixed_costsoptionalFloat0Unit: Dollars. One-time fixed cost of activating a vehicle. Example: 100 implies $100 is charged for using this vehicle
distance_dependent_costsoptionalFloat0Unit: Dollars/meter. Cost incurred per meter travelled by vehicle. Suitable for quantifying fuel cost. Example: 1 implies $1 is charged for 1 meter travelled by vehicle
time_dependent_costsoptionalFloat0Unit: Dollars/second. Cost incurred per second spent by vehicle travelling. Example: 1 implies $1 is charged for 1 second spent by vehicle travelling
waiting_time_dependent_costsoptionalFloat0Unit: Dollars/second. Cost incurred per second spent by driver waiting to serve the customer. Example: 1 implies $1 is charged for 1 second spent by driver waiting to serve customer.

time_windowsrequiredTimeWindow[]List of time windows (i.e. time when customer is available to collect or pickup items)
