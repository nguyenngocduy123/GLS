POST /web/user/v2

Create a user

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RoleAdmin
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
usernamerequiredStringUsername of user
rolerequiredStringRole of user. Allowed values: PLANNER, DRIVER, DEFAULT, ADMIN
emailrequiredStringEmail address of user
fullnameoptionalStringFull name of user
usergroupoptionalStringUsergroup assigned to the user
nricoptionalStringUnique identifer of user
phoneoptionalStringContact number of user
noteoptionalStringRemarks/memo about user
disabledoptionalBooleanIndicates whether account can be used

Request Example

    // http://localhost/web/user/v2
    {
        "username": "user_doe",
        "role": "planner",
        "email": "example@email.com",
        "fullname": "User Doe",
        "usergroup": null,
        "nric": null,
        "phone": null,
        "note": null,
        "disabled": false
    }

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
messagerequiredResponse message

Success Response Example

    // http://localhost/web/user/v2
    {
        "message": "user_doe has been registered successfully"
    }
