POST /web/usergroup/v2

Create usergroup(s). Usergroups will be created as many as possible (i.e. if any fails, other usergroups will still be persisted)

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Body

FieldRequiredTypeDefault ValueDescription
[]requiredObject[]List of usergroups to create
[].usergrouprequiredStringName of usergroup
[].descriptionoptionalStringDescription of usergroup

Request Example

    // http://localhost/web/usergroup/v2
    [{
        "usergroup": "usergroup1",
        "description": null
    }]

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
[]requiredObject[]List of inserted usergroups
[]._idrequiredStringUsergroup identifier
[].usergrouprequiredStringActual name of usergroup
[].descriptionoptionalStringDescription of usergroup
[].modified_daterequiredStringDate that the usergroup was last modified

Success Response Example

    // http://localhost/web/usergroup/v2
    [{
        "_id": "5bac93582c431fbef17d4b08",
        "usergroup": "USERGROUP1",
        "description": null,
        "modified_date": "2018-01-01T09:00:00Z"
    }]

DELETE /web/usergroup/v2

Delete usergroup(s). Usergroups will be created as many as possible (i.e. if any fails, other usergroups will still be persisted)

Resource Info

Information
Response formatText
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Query String

FieldRequiredTypeDefault ValueDescription
usergroupsrequiredStringComma-delimited string. List of usergroups

Request Example

  • Delete usergroups
    • http://localhost/web/usergroup/v2?usergroups=USERGROUP1

Success Response

HTTP Status 200

FieldTypeDefault ValueDescription
IntegerNumber of deleted usergroups

Success Response Example

    // HTTP/1.1 200 OK
    1 // one record was updated

GET /web/usergroup/v2

Get list of usergroups

Resource Info

Information
Response formatJSON
Requires authentication?Yes
Authorised RolePlanner
Rate limited?No
CSRF enabledYes

Request Query String

FieldRequiredTypeDefault ValueDescription
usergroupsoptionalStringComma-delimited string. List of usergroups to strictly query on
projectionoptionalStringComma-delimited string. List of fields to request to return

Request Example

  • Get all usergroups
    • http://localhost/web/usergroup/v2
  • Get usergroups that are within a list
    • http://localhost/web/usergroup/v2?usergroups=USERGROUP1
  • Get usergroup details with additional field
    • http://localhost/web/usergroup/v2?projection=description

Success Response

HTTP Status 200

FieldRequiredTypeDefault ValueDescription
[]requiredObject[]List of inserted usergroups
[]._idrequiredStringUsergroup identifier
[].usergrouprequiredStringActual name of usergroup
[].descriptionoptionalStringDescription of usergroup. Not projected by default
[].modified_daterequiredStringDate that the usergroup was last modified

Success Response Example

    // http://localhost/web/usergroup/v2
    [{
        "_id": "5bac93582c431fbef17d4b08",
        "usergroup": "USERGROUP1",
        "modified_date": "2018-01-01T09:00:00Z"
    }]
